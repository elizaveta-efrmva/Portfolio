# Чеклист видеопортфолио

## Локальная проверка

- [ ] `npm run dev` запускает сайт без ошибок.
- [ ] `npm run lint` проходит без ошибок.
- [ ] `npm run build` создает папку `out/`.
- [ ] В `package.json` нет Notion-зависимостей.
- [ ] В `public/` нет `notion-images`.

## Контент

- [ ] Имя, роль, заголовки и контакты заменены в `content.config.ts`.
- [ ] Все работы описаны в `content.config.ts`.
- [ ] Для каждой работы задана корректная ориентация: `horizontal` или `vertical`.
- [ ] Постеры лежат в `public/posters/`.
- [ ] Источники видео заменены с `PLACEHOLDER` на `embed`, `file` или `link`.

## Интерфейс

- [ ] Header, якорная навигация и мобильное меню работают.
- [ ] Hero показывает шоурил или заглушку.
- [ ] Галерея фильтруется по категориям.
- [ ] Вертикальные и горизонтальные карточки сохраняют правильные пропорции.
- [ ] Модалка открывает плеер, закрывается по Esc и клику вне.
- [ ] Фокус видим на интерактивных элементах.

## Адаптив

- [ ] 360px: нет горизонтального скролла, меню работает.
- [ ] 768px: сетка и карточки не ломаются.
- [ ] 1280px: hero, галерея, about, services и contact выглядят цельно.

## Деплой

- [ ] В GitHub Pages выбран Source: GitHub Actions.
- [ ] Для repo-pages задан `BASE_PATH=/<repo>`.
- [ ] Для user-pages `BASE_PATH` пустой.
- [ ] Workflow `Deploy to GitHub Pages` проходит успешно.
- [ ] Ссылки на постеры и видео работают на опубликованном URL.
